## Descripción del sistema

Este proyecto consiste en un sistema básico de gestión de restaurante desarrollado en Python utilizando Programación Orientada a Objetos (POO). El sistema permite registrar productos y clientes, almacenarlos en listas y mostrar la información organizada en consola.

## Estructura del proyecto

```text
Proyecto_Restaurante
│
├── restaurante_app
│   ├── modelos

│   │   ├── producto.py
│   │   └── cliente.py
│   │
│   ├── servicios
│   │   ├── __init__.py
│   │   └── restaurante.py
│   │
│   └── main.py
│
└── README.md
```

## Tipos de datos utilizados

* str: nombre, correo.
* int: edad, cantidad.
* float: precio.
* bool: disponible, activo.
* list: almacenamiento de productos y clientes.

## Funcionalidades

* Registrar productos del restaurante.
* Registrar clientes.
* Almacenar información mediante listas.
* Mostrar información organizada en consola.
* Utilizar importaciones entre módulos.

## Reflexión

El uso de identificadores descriptivos facilita la lectura y comprensión del código. Los tipos de datos adecuados permiten representar correctamente la información del sistema, mientras que las listas ayudan a almacenar múltiples objetos de forma organizada. La programación modular mejora el mantenimiento y reutilización del código.
